Zadatak: Prodavnica piva (korišćenjem radnih okvira Spring Boot, AngularJS i Bootstrap razviti Web aplikaciju za
vođenje evidencije prodavnice piva. Aplikacija treba da obezbedi rad sa sledećim entitetima: piva, pivara, vrsta piva)

Uradjeno:
	- CRUD logika sva tri entiteta (pivo, pivara i vrsta piva)
	
	- Pomoću radnog okvira Spring Boot implementirani sledeći REST API:
		● GET /api/pivare - preuzimanje svih pivara
		● GET /api/pivare/{id} - preuzimanje pivare po zadatom id-u
		● GET /api/piva - preuzimanje svih piva (paginirano)
		● GET /api/piva/{id} - preuzimanje piva po zadatom id-u
		● POST /api/piva - dodavanje novog piva
		● PUT /api/piva/{id} - izmena postojećeg piva
		● DELETE /api/piva/{id} - brisanje postojećeg piva
		● GET /api/pivare/{id_pivare}/piva - preuzimanje svih piva za zadatu pivaru
		● GET /api/vrste_piva - preuzimanje svih vrsta piva
	
	- Na stranicu za prikaz svih piva dodato je dugme "Kupi".
		Nakon klika na ovo dugme, ukoliko je količina na stanju veća od nule, količina se
		umanjuje za jedan, a korisnik dobija poruku o uspešnoj kupovini i prikaz na stranici se
		osvežava. Ukoliko je stanje 0, korisnik dobija poruku da nije moguće izvršiti
		kupovinu i prikaz na stranici se osvežava.
		
	- Obezbedjeno je dodavanje, brisanje kao i izmena na zasebnoj stranici (takodje je omogucena izmena na tekucoj stranici,
		nakon klika na dugme izmeni popunjava se forma sa zeljenim podacima, prilikom cuvanja podataka tabela se osvezava)
		
	- Omogucen je paginirani prikaz podataka u tabeli.
